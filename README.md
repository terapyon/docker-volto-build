# volto-build

This Docker is to build for Volto.

## Volto version

"@plone/volto": "5.7.1"

## Docker build and test

```
$ docker build -t local/volto-build .
$ docker run -it --rm local/volto-build
```

## Run

```
$ docker run -it --rm terapyon/volto-build
```
